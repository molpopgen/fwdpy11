class fwdpy11.GaussianNoise(sd: float, mean: float = 0.0)¶

Gaussian noise added to genetic values. This class has the following attributes, whose names are also kwargs for intitialization. The attribute names also determine the order of positional arguments:

Parameters

  • sd (float) – Standard deviation

  • mean (float) – Mean value
